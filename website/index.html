Project goal

Cloud assistants are capable, but they send your data elsewhere. Ollama Chat targets optimal local intelligence in the ~9B class — the best trade-off between quality, speed, and RAM on consumer hardware.

Local-first

Ollama on localhost. Conversations stay on disk and in your browser — not a vendor's server.

9B curated

Pre-configured for Qwen 3.5 9B and GLM-4 9B — compare models side-by-side in one click.

Production-like

HTTPS UI, streaming responses, file attachments, persistent history, server controls.

Default model stack

primary

Qwen 3.5 9B

qwen3.5:9b

alternate

GLM-4 9B

glm4:9b

Complementary bilingual reasoning style at the same parameter scale.

Select all models in the app to stream answers from every installed model simultaneously.

Install

1

Clone & install

git clone https://github.com/avinashpeyyety/ollama-chat-app.git
cd ollama-chat-app
npm install
2

Install Ollama + 9B models

brew install ollama && brew services start ollama   # macOS
npm run install-models   # pulls qwen3.5:9b + glm4:9b
3

Launch

npm start
# open https://localhost:3443

One-liner setup: npm run setup:full — certificates + model downloads.
